## Getting Started
### Prequisites
- UnityHub Installed
- Unity version 6000.0.68f1 or newer
- Git installed or download the zip

### Clone or Download the Repo
With Git:

    git clone https://github.com/OsmanAga31/GREYFIRE

Or download as ZIP: Go to the repository page → click Code → Download ZIP → extract it to a folder of your choice.

### Add Project to Unity

 1. Open UnityHub
 2. Click "Add" → "Add Project from disk"
 3. Navigate to the folder where you cloned/extracted the project and select it
 4. Click "Open"

> If you don't have the correct Unity version installed, UnityHub will ask you to install it automatically

### Play the Game

 1. If not already, open the "Project" panel/tab *(File Explorer in Unity)*
 2. Navigate to "Assets/Scenes/" and double-click the scene "Level-1" to open it.
 3. Press the Play button at the top of the Editor
 4. Have Fun! 

### Alternative Download

 1. Press the "Releases" button on the right sigh of the Repository page
 2. If necessary: Click "Assets" to expand
 3. Click GREYFIRE.zip
 4. Extract the ZIP to a folder of your choice
 5. Double click "GREYFIRE.exe" in the "Build" folder
 6. Have fun!
 7. Press "Alt + F4" to exit the game :)
